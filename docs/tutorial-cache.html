Documentation: Caching

Caching

The object manager has a option to use a json file as a module resolution cache. This helps to improve speed in very large setups. But more importantly, it helps you to get an overview, what the injection map in the background looks like. The file is formatted json and should be readable by anyone who knows, how json looks like.

example
